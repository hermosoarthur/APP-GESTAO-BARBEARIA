
# Gestão de Barbearia

App para gestão de clientes, datas e serviços oferecidos em uma barbearia. App este voltado para o público unissex e para quem planeja melhorar a gestão de seu comércio.  


## Objetivos
Utilizar os aprendizados em aula em todo o app para otimização e facilitação do uso do app para a gestão por parte dos utilizadores que são os barbeiros
## Instalação

Pré-requisitos:
- Node.js: Instalado no seu computador.
- Expo CLI: Instalado globalmente.
- App Expo Go: Instalado no seu celular (Android ou iOS).

```bash
  npm install -g expo-cli
  npm install APP-GESTAO-BARBEARIA
  cd APP-GESTAO-BARBEARIA
  expo start
```

    
### Abordagem
Decidimos utilizar as stacks aprendidas em aula junto às ferramentas recomendadas pelo professor, a fim de colocar em prática todo o conteúdo do semestre em um único projeto. O objetivo é desenvolver um aplicativo comercial que possa ser aprimorado ao longo do tempo, permitindo a implementação de novas funcionalidades e tecnologias. Dessa forma, buscamos ampliar nossos conhecimentos sobre novas tendências e nos manter atualizados.
## Stack utilizada

**Front-end:** React

**Back-end:** Node, Firebase
